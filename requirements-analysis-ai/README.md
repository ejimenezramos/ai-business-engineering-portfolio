# AI Requirements Analysis

This project demonstrates how Artificial Intelligence and Large Language Models (LLMs)
can be applied to a common real-world scenario in software engineering and consulting:
translating business requirements into actionable technical work.

The application takes business requirements written in natural language and analyzes
them to extract structured technical information.
